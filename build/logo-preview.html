📦 Export Instructions

To use this logo in the Electron app, you need to export it to different formats:

Option 1: Using Inkscape (Free)

# Install Inkscape
brew install inkscape

# Export to PNG (all sizes)
inkscape icon.svg --export-filename=icon.png --export-width=1024 --export-height=1024

Option 2: Using Online Tools

1. Go to convertio.co/svg-png

2. Upload icon.svg

3. Download PNG at 1024×1024

Option 3: macOS App Icon Generation

# After having icon.png (1024×1024), create .icns for macOS
mkdir icon.iconset
sips -z 16 16 icon.png --out icon.iconset/icon_16x16.png
sips -z 32 32 icon.png --out icon.iconset/icon_16x16@2x.png
sips -z 32 32 icon.png --out icon.iconset/icon_32x32.png
sips -z 64 64 icon.png --out icon.iconset/icon_32x32@2x.png
sips -z 128 128 icon.png --out icon.iconset/icon_128x128.png
sips -z 256 256 icon.png --out icon.iconset/icon_128x128@2x.png
sips -z 256 256 icon.png --out icon.iconset/icon_256x256.png
sips -z 512 512 icon.png --out icon.iconset/icon_256x256@2x.png
sips -z 512 512 icon.png --out icon.iconset/icon_512x512.png
sips -z 1024 1024 icon.png --out icon.iconset/icon_512x512@2x.png
iconutil -c icns icon.iconset
rm -rf icon.iconset

Required files for Electron:

  • build/icon.png — 1024×1024 PNG (used by electron-builder)
  • build/icon.icns — macOS app icon
  • build/icon.ico — Windows app icon (if supporting Windows)
